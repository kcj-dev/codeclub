About

Code Club Canada

At Code Club Canada, we think all children should have the opportunity to learn to code.

To do this, we support a nationwide network of volunteers and educators who run free coding clubs for children aged 8-12 to build and share their ideas and learning along the way. Code Club Canada is a project with the Montreal-based national non-profit organization, Kids Code Jeunesse.

ABOUT CODE CLUB INTERNATIONAL

Founded in 2012, Code Club International currently has over 100 countries, and our club projects have been translated into 28 languages.

Our projects are step-by-step guides which teach children learn Scratch, HTML, Python & and more! There’s no need to be a computing expert, the projects gradually introduce coding concepts by making games, animations, and websites that adults and children can easily follow along.

Code Clubs are a key component to educational programs with  local partners in 12 countries. In 2015 we joined forces with the Raspberry Pi Foundation to bring even more opportunities for young people to learn how to code.
